# Installation

## Requirements

- Node.js 18 or later
- A monorepo (pnpm, npm, yarn — any workspaces setup works)
- Claude Code CLI installed (`npm install -g @anthropic-ai/claude-code` or the native install)

## Install the package

```bash
# npm
npm install -D hivescan

# pnpm
pnpm add -D hivescan

# yarn
yarn add -D hivescan
```

## Run it

```bash
npx hivescan
```

Or install globally and run:

```bash
npm install -g hivescan
hivescan --dir ~/Sites
```

HiveScan will auto-discover projects in the target directory, start the server, and open `http://localhost:4269` in your browser.

```
  Scanning: /Users/you/Sites
  HiveScan running at http://localhost:4269
  12 project(s) found, scanning ports every 5s...
```
