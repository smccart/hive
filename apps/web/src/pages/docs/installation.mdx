# Installation

## Requirements

- Node.js 18 or later
- A monorepo (pnpm, npm, yarn — any workspaces setup works)
- Claude Code CLI installed (`npm install -g @anthropic-ai/claude-code` or the native install)

## Install the package

```bash
# npm
npm install -D hive-agents

# pnpm
pnpm add -D hive-agents

# yarn
yarn add -D hive-agents
```

## Add a script

Add a `hive` script to the **root** `package.json` of your monorepo:

```json
{
  "scripts": {
    "hive": "hive"
  }
}
```

## Run it

```bash
npm run hive
```

HiveAgents will auto-discover `apps/*/` and `packages/*/` directories, start the server, and print:

```
  HiveAgents: auto-discovered 4 agent(s) from apps/ and packages/
  Create hive.config.js to customize.

  HiveAgents running at http://localhost:4199
```

Open [http://localhost:4199](http://localhost:4199) in your browser. You'll see your agents listed in the sidebar.

## Next step

Create a `hive.config.js` to customize which directories appear, their labels, colors, and port links — see [Configuration](/docs/configuration).
