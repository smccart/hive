# HiveScan

A web-based terminal dashboard for managing multiple [Claude Code](https://claude.ai/code) agents across your projects.

## Quick Start

```bash
npx hivescan
```

Or install globally:

```bash
npm install -g hivescan
hivescan --dir ~/Sites
```

HiveScan will scan your current directory for projects, start a local dashboard, and open it in your browser.
